# Week 2 Week-Local Code

Use this folder for week-local reusable helpers. If the logic becomes useful
across multiple weeks, move it into `fintools/`.

Current week-local modules include:

- `market_panel.py` for the shared FRED month-end teaching panel
- `fred_stage1.py` for the Week 2 FRED Stage 1 timing contract and long-table builder
- `australia_macro_panel.py` and `australia_macro_specs.py` for the Australia
  mixed-frequency pipeline
- `beginner_plotting.py` for the Week 2 synthetic time-series plotting on-ramp
